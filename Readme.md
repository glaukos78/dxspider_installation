With this script you can install and configure in the few minutes ths DxSpider Cluster

Installation

    1. Download script.
    2. Must be run as root user.
    3. Change permission
    chmod + x setup_dxspider.sh
    4. Run script and follow the messages.


Script has been tested 
    1. CentOS 7
    2. Raspbian 7
    3. Raspbian 8
